# LLMI-CDP Seminar Slides

Deck trình bày paper:

**A large language model for multimodal identification of crop diseases and pests**  
Wang et al., Scientific Reports 2025, 15:21959.

## Cách mở

Mở `index.html` trong trình duyệt hoặc chạy một static server trong thư mục này.

## Điều khiển slide

- Mũi tên phải / Space: slide tiếp theo.
- Mũi tên trái: quay lại.
- Home / End: slide đầu / cuối.
- In hoặc xuất PDF: dùng chức năng Print của trình duyệt.

## Kích thước

Deck dùng canvas **1920 × 1080** và tự scale theo kích thước cửa sổ trình duyệt.
